# Shopify Theme - CTB Garage Black Mode

This is the CTB Garage auto body shop Shopify theme with dark/black mode styling.

## Theme Sections

- **ctb-hero** - Video slider hero with animated title and CTA buttons
- **ctb-hero-features** - Feature list with icons
- **ctb-about** - About section with feature list and image slider
- **ctb-testimonials** - Customer testimonials carousel
- **ctb-gallery** - Work/project gallery carousel
- **ctb-merch** - Merchandise product carousel
- **ctb-faq** - Accordion FAQ section
- **ctb-contact** - Contact info and form section

## Development

This theme uses:
- Splide (carousel library): https://splidejs.com/
- Font Awesome 6 (icons): https://fontawesome.com/

### Setup

```bash
npm install -g @shopify/cli
shopify theme dev
```

### Deploy

```bash
shopify theme push
```

## Color Scheme

- Background: #050505 (near black)
- Text: #f5f5f5 (light)
- Accent: #f5d76e (gold)
